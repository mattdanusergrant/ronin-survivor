# Ronin Survivor — artist-reference prompts (vol. 1)

Ten image-generation prompts, one per entity in this folder. Each is
self-contained — paste into Midjourney, DALL-E, SDXL, or any other
text-to-image model and it should produce a usable reference image
your artist can riff on.

The prompts share a deliberate house style (limited palette, ink-brush
silhouette feel, near-black background, crisp pixels) so the outputs
read as one set instead of ten unrelated sketches.

**Suggested model params**

- Midjourney: append `--ar 1:1 --stylize 100 --v 6`
- DALL-E 3: set style to "natural", size 1024×1024
- SDXL: 768×768, CFG ~7, sampler DPM++ 2M Karras, add negative prompt
  `blurry, anti-aliased, smooth gradients, painterly, 3d render`

---
